# LaunchPad - Interview Preparation Platform

LaunchPad is a modern interview preparation platform designed for students to practice aptitude, coding, and HR interview questions.

## Features
- Aptitude MCQ Practice
- Coding Lab
- HR Interview Questions
- Full Mock Tests
- Progress Tracking Dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
